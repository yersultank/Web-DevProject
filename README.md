# AssetTrack

Simple web app for tracking company assets, assignments, and condition reports.

## Team Members

- Yersultan
- abnzrdev
- yersultank

## How to Start

From the project root:

1. Start backend:
   - `./scripts/start-backend.sh`
2. Start frontend:
   - `./scripts/start-frontend.sh`

Or run both in separate terminals automatically:

- `./scripts/start-project.sh`

## Seed Demo Data

Run:

- `./scripts/seed-data.sh`

Demo accounts:

- Admin: `admin` / `pass1234`
- Users: `user1`, `user2`, `user3` / `pass1234`
